## 1. 專案概述 (Project Overview)
**WVC-plan (Worker View Calendar)** 是一個輕量化的、基於單一 HTML 檔案的工班管理工具。
- **核心目標**：解決工班排班視覺化、考勤紀錄智能化與任務管理便捷化的需求。
- **技術架構**：採用 React + Tailwind CSS (經由 CDN 載入)，實現「開啟即用、無需安裝」的純前端應用。資料存儲於瀏覽器 LocalStorage 或透過 JSON 檔案進行匯入/匯出。

## 2. 核心功能模組 (Core Modules)

### 2.1 視覺化月曆排班 (Main Calendar)
- **視覺呈現**：提供 2026 年國定假日、農曆與週末標記的完整月曆。
- **即時統計**：每日格子下方顯示當日「出勤」與「未出勤」人數統計。
- **人員管理**：支援從 Excel 匯入工班名單，並可點擊人員進行姓名、簡稱修改與「隱藏」設定。
- **歷史保全**：實作了「人員歷程架構 (Workers History)」，人員屬性的變更僅影響今天及未來，確保過去的歷史記錄不被覆蓋。

### 2.2 任務精靈與料件管理 (Task Wizard)
- **快速新增**：透過「業主 -> 標案 -> 區域 -> 地址」的引導式介面 (Wizard) 快速建立每日任務。
- **待辦工作**：內建從 Word 報表分析出的「未完工程」清單，可以一鍵轉為當日任務。
- **料件清單**：每日可設定需使用的料件（標線機、原料、數量等）。

### 2.3 智能化考勤分析 (Attendance Analyzer)
- **OCR 辨識**：支援貼上打卡系統截圖，利用 Tesseract.js AI 辨識員工姓名與打卡時間。
- **精準權重 (Name Recognition Weighting)**：針對易錯字（如：廖旻揆、林宏叡、普萬托）設定了專屬的特徵字權重加分機制，大幅提高 OCR 辨識準確率。
- **異常判定**：自動比對基準上班時間，計算遲到分鐘數，並標記「下班未打卡」等異常狀態。
- **Excel 支援**：支援直接讀取考勤系統匯出的 Excel 報表，達到 100% 精準分析。

### 2.4 車輛軌跡分析 (Track Analyzer)
- **數據彙整**：讀取車隊管理系統的軌跡 Excel 檔。
- **自動計算**：自動推算總里程、行駛時間、靜止怠速時間。
- **停留點監測**：自動篩選出停留超過 10 分鐘的地點、時間與座標，方便核對出勤真實性。
