# 更新日誌 (Changelog)

所有關於 WVC-plan (工班出勤排班表) 的重要更新與版本紀錄將會在此檔案中維護。

## [1.2.0] - 2026-03-06

### 新增 (Added)
- 實作「任務精靈 (Task Wizard)」與 Excel 任務清單匯入功能。
- 現在可點擊「匯入任務清單」讀取 xlsx (需包含業主、標案欄位)，讓新增每日任務時可以用按鈕點擊「業主 -> 標案 -> 縣市區域 -> 手寫地址」的方式快速完成。
- 修改原本的純文字輸入框，並將「標案 (Project)」欄位加入任務清單中。

## [1.1.0] - 2026-03-06

### 新增 (Added)
- 實作「每日任務」管理功能：可在每個日期加入多筆任務（包含業主、區域、地址、備註欄位）。
- 實作「使用料件」管理功能：可在每個日期加入多筆料件需求（包含標線工具、原料、數量欄位）。
- 任務與料件將會被記錄於 JSON 存檔中，並於月曆格子的下方直接顯示。

## [1.0.1] - 2026-03-06

### 變更 (Changed)
- 將主程式 `wvc-1.html` 更名為 `index.html`，以便未來部署至 GitHub Pages 時可以直接透過網址讀取。

## [Initial Version] - 2026-03-06

### 新增 (Added)
- 建立專案基礎架構，使用純前端技術 (HTML + React CDN + Tailwind CDN)。
- 建立 `wvc-1.html` 主程式。
- 實作視覺化月曆排班介面，包含周末、假日紅字與農曆提示。
- 實作點擊日期彈出 Modal 單獨設定人員排班狀態的功能。
- 實作匯入 Excel (`.xlsx`) 自動解析「工班清單」與自訂「狀態顏色」功能。
- 實作「文字輸入排班」解析器，支援多行文字智慧轉譯班表。
- 實作匯出 JSON 備份檔及匯入 JSON 還原功能。
- 實作利用 `html2canvas` 與 `jsPDF` 將月曆轉換並匯出橫版 A4 PDF 的功能。
