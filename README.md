# WVC-plan (工班出勤排班表)

這是一個基於單一 HTML 檔案搭配 React 與 Tailwind CSS 構建的輕量化**工班排班管理工具**。可以在瀏覽器中直接開啟使用，無需依賴後端伺服器，資料全數存放於本地。

## 🌟 主要功能

1. **視覺化排班月曆**
   - 直覺的月曆介面，清楚顯示每日出勤與未出勤人數統計。
   - 自動帶入 2026 國定假日與農曆日期標示。
   - 點擊日期即可跳出彈窗編輯當日人員排班狀態。

2. **自訂狀態與人員名單 (Excel 匯入)**
   - 支援將包含「工班清單」與「狀態圖例」的 Excel (`.xlsx`) 檔案匯入，動態更新總編制與狀態顏色標示。

3. **文字快速排班功能**
   - 支援貼上特定格式的文字（包含日期、時間、人員簡稱），系統會自動解析並分配正確的班表狀態。
   - 自動處理未知人員代號對應至「點工」。

4. **資料本地端儲存與讀取 (JSON)**
   - 允許把目前設定好的當月排班保存為 `.json` 格式下載至電腦。
   - 後續可隨時讀取 `.json` 檔案回復排班狀態。

5. **匯出排班表 PDF**
   - 一鍵將視覺化排班表匯出成 A4 橫向 PDF，方便列印、截圖或傳送給團隊。

## 🚀 使用方式

1. 將本專案 clone 到本地。
2. 直接雙擊開啟 `wvc-1.html`，即可在網頁瀏覽器中操作。
3. （可選）透過匯入 Excel 初始化您的團隊人員名單與自訂的上下班狀態。

## 🛠 技術堆疊

- HTML5 / CSS3
- JavaScript (ES6+ / Babel)
- **React (CDN 匯入)**
- **Tailwind CSS (CDN 匯入)**
- **xlsx** (解析 Excel 檔案)
- **html2canvas & jsPDF** (產生 PDF)
